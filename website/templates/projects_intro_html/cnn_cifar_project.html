Object Recognition System

CNN on CIFAR-10 Dataset

The Project is developed over the CIFAR-10 dataset. This dataset consists of color images of the 10 different objects namely car, ship, dog, cat etc. The project was developed by using one of the deep learning method- "Convulational Neural Network".

The project is a web appplication that lets the user upload the images and then recognizes which object is it.
