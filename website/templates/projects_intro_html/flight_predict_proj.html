Flight Fare Prediction

Random Forest Regression

The Project is developed over the 10000 flights fare dataset. The flights are between the metro cities. The project was developed by using one of the core machine learning algorithm- "Random Forest Regression". Lots of feature engineering concepts were involved in reaching to the final datapoint. Methods like One Hot Encoding, Ordinal Encoding, date time bifurcations were used in the project. Hyperparameter tuning was also performed to improve the accuracy.

The project is a web appplication that predicts the price on the flights.
