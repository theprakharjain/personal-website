Number Recognition System

CNN on MNIST Dataset

The Project is developed over the MNIST dataset. This dataset consists of greyscale images of the numbers ranging between 0-9. The project was developed by using one of the deep learning method- "Convulational Neural Network".

The project is a web appplication that lets the user draw the images and then recognizes which number is it.
